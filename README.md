# Customer Gross Margin Analysis using Microsoft Excel

## Project Overview

This project demonstrates how Microsoft Excel can be used to analyze sales data and visualize customer gross margin performance.

The analysis uses a sales dataset to calculate the total gross margin generated by each customer and presents the results using a column chart for easy comparison.

---

## Objective

- Analyze customer gross margin.
- Compare customer profitability.
- Present the results using an Excel chart.

---

## Dataset

The workbook contains:

- Raw sales data
- Customer information
- Gross Margin values

---

## Analysis Performed

- Reviewed sales data
- Summarized gross margin by customer
- Created a column chart to compare customer gross margins

---

- Microsoft Excel
- Excel Charts
